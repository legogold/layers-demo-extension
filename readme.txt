Demo Layers Extension for Layers WP
=============
Author: Vail Joy
URL: http://docs.layerswp.com/


This demo extension requires the Layers theme to be installed, which you can download free!
http://www.layerswp.com/download/layers/



## Install

Extract this file to the wp-content/plugins dir of your setup


## Help & Documentation

This extension is intended as a teaching tool. We strongly recommend reading through the commented links, or the entire development guide beginning here:
http://docs.layerswp.com/plugin-creation-guide/
